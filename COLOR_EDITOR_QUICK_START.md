# Template Color Editor - Quick Start Guide 🚀

## Get Started in 30 Seconds!

---

## 🎯 How to Access

### Step 1: Open the Editor
- Go to `/customize` page
- You should see the business card editor

### Step 2: Click Color Tab
- Look at the **left sidebar**
- Find the **Color icon** (droplet/palette icon)
- It's the **6th icon** from the top
- Click it!

```
Left Sidebar Icons:
1. Text
2. Uploads
3. Graphics
4. Background
5. Templates
6. 🎨 Color ← CLICK HERE!
7. More
```

---

## 🎨 Using the Color Editor

### Step 1: Browse Color Schemes
You'll see **12 professional color schemes** in a grid:

```
Forest Green   Ocean Blue    Royal Purple
Sunset Orange  Ruby Red      Emerald Teal
Golden Yellow  Midnight Navy Rose Pink
Slate Gray     Bronze Brown  Mint Green
```

Each scheme shows a **4-color preview** (2x2 grid).

---

### Step 2: Apply a Color Scheme

**Method 1: Click Anywhere on Card**
- Click any color scheme card
- Colors update instantly!

**Method 2: Hover First**
- Hover over a scheme (card scales up)
- Click to apply

---

### Step 3: See the Results

**What Happens:**
1. ✅ Background color changes
2. ✅ All text colors update
3. ✅ All shape colors update
4. ✅ All borders update
5. ✅ Success notification appears
6. ✅ Checkmark shows on selected scheme

**Toast Message:**
```
✓ Applied Forest Green color scheme!
```

---

### Step 4: Reset (Optional)

**Don't like the new colors?**
- Click **"Reset"** button (top right)
- Colors return to original
- Selection clears

## 🆘 Troubleshooting

### Colors not changing?
- Make sure you clicked the scheme card
- Check if elements are locked
- Try clicking Reset and reapplying

### Don't see Color tab?
- Make sure you're in the editor (/customize)
- Look for the droplet/palette icon
- It's the 6th icon in left sidebar

### Colors look wrong?
- Try a different scheme
- Click Reset to undo
- Some templates work better with certain schemes

### Want original colors back?
- Click "Reset" button
- Or reload the page (if not saved)
- Or load the original template again
