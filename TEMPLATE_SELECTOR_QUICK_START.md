# Template Selector - Quick Start Guide 🚀

## What's New?

A beautiful, professional template selector modal has been added to the QuickCard editor! Now you can browse, filter, and switch templates with ease.

---

## 🎯 How to Access

### Method 1: Change Template Button (NEW!)
1. Open the editor at `/customize`
2. Look at the top action bar
3. Click the **purple "Change Template"** button
4. The template selector modal opens

```
Top Bar: [←] [Design Name] | [↶] [↷] | [Safety] [Bleed] | [📋 Change Template] ← Click here!
```

### Method 2: Templates Tab (Existing)
1. Open the editor at `/customize`
2. Click the **Templates icon** in the left sidebar (6th icon)
3. Browse templates in the sidebar panel
4. Click any template to load it

---

## 🎨 Using the Template Selector

### Step 1: Open the Modal
Click the **"Change Template"** button in the top action bar.

### Step 2: Browse Templates
The modal shows all available templates in a beautiful grid layout:
- **Template preview** - See the design
- **Template name** - Identify the template
- **Description** - Understand the style
- **Color dots** - See color variations

### Step 3: Filter Templates (Optional)

#### Filter by Industry:
```
Left Sidebar → Industry Section
○ Agriculture & Farming
○ Animals & Pet Care
○ Arts, Crafts, and Design
○ Automotive & Transportation
○ Beauty & Spa
○ Construction & Contractors
○ Education & Training
○ Food & Restaurant
○ Health & Medical
○ Legal & Financial
○ Real Estate
○ Technology & IT
● All Industries
```

#### Filter by Color:
```
Left Sidebar → Design colour Section
🔵 🟢 🟡 🟠 🔴 ⚪ ⚫ 🟤 🟣 🩷 🟧
Click any color to filter templates
```

#### Filter by Size:
```
Left Sidebar → Size & orientation Section
○ Horizontal (9cm × 5.2cm)
○ Vertical (5.2cm × 9cm)
```

### Step 4: Search Templates (Optional)
```
Top of Modal → Search Bar
🔍 Search designs
Type: "medical", "modern", "corporate", etc.
```

### Step 5: Select a Template
1. **Hover** over any template card
2. Blue overlay appears
3. **"Use Template" button** shows
4. **Click** the button or anywhere on the card

### Step 6: Template Loads
1. Canvas clears automatically
2. New template loads with all elements
3. Success notification appears: ✓ Template "Name" loaded!
4. Modal closes
5. Start customizing!

## 💡 Tips & Tricks

### Quick Selection:
- **Click anywhere** on the template card to select it
- No need to wait for hover - just click!

### Clear Filters:
- If no templates show, click **"Clear filters"** at the bottom
- Resets all filters to show all templates

### Keep Current Design:
- Changed your mind? Click **"Current template"** button in footer
- Modal closes without changing your design

### Close Modal:
- Click the **X button** (top right)
- Click **outside the modal** (on the backdrop)
- Press **Escape key**

### Search Tips:
- Search by template name: "Graphic Mitra"
- Search by style: "modern", "vintage", "medical"
- Search by category: "creative", "professional"

## 🎨 Customizing After Loading

Once a template loads, you can customize everything:

### Text:
1. **Double-click** any text to edit
2. Use **floating toolbar** to style (font, size, color, etc.)
3. Use **quick actions** below text (font size shortcuts)

### Graphics:
1. **Click** any shape or image to select
2. Use **sidebar editor** to customize (colors, size, rotation, etc.)
3. **Drag** to reposition

### Background:
1. Go to **Background tab** in sidebar
2. Choose solid color or gradient
3. Click to apply

### Add Elements:
1. **Text tab** - Add new text fields
2. **Graphics tab** - Add shapes, icons, illustrations
3. **Uploads tab** - Add your own images

## ⚠️ Important Notes

### Template Loading:
- ⚠️ **Loading a template clears your current design**
- Make sure to **save your work** before switching templates
- There's no undo for template changes (yet)

### Auto-Save:
- If editing an existing design, it auto-saves every 3 seconds
- New designs need to be manually saved first
- Template changes are not auto-saved until you make edits

### Browser Support:
- Works best in **Chrome, Firefox, Safari, Edge** (latest versions)
- Requires JavaScript enabled
- Requires modern browser (ES6+ support)

## 🆘 Need Help?

### Template not loading?
- Check your internet connection
- Refresh the page
- Try a different template
- Check browser console for errors

### Can't find a template?
- Clear all filters
- Try different search terms
- Check if you're in the right category

### Modal won't open?
- Make sure you're logged in
- Refresh the page
- Check if JavaScript is enabled
- Try a different browser

### Lost your design?
- Check "My Designs" page
- Look for auto-saved versions
- Contact support if needed
