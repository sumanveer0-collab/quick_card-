## 🎨 Features

### **1. Floating Toolbar (Upar)**
```
┌────────────────────────────────────┐
│ [Arial ▼] [-] [48] [+] [B][I][U] │
└────────────────────────────────────┘
```

**Kya Hai:**
- Font dropdown (Arial, Poppins, etc.)
- Font size with +/- buttons
- Bold, Italic, Underline buttons

**Kab Dikhta Hai:**
- Jab text select karo
- Element ke **upar** float karta hai
- Auto-position hota hai

---

### **2. Quick Actions (Neeche)**
```
┌──────────────────────┐
│ [+] [-] [🗑] [⧉] [🔒]│
└──────────────────────┘
```

**Kya Hai:**
- **[+]** - Font size badhao
- **[-]** - Font size ghatao
- **[🗑]** - Delete karo
- **[⧉]** - Duplicate karo
- **[🔒]** - Lock/Unlock karo

**Kab Dikhta Hai:**
- Jab text select karo
- Element ke **neeche** float karta hai

## 🚀 Kaise Use Karein

### **Step 1: Text Select Karo**
```
1. Text pe click karo
2. Cyan border dikhega
3. Circular handles dikhenge
4. Toolbar upar dikhega
5. Actions neeche dikhenge
```

### **Step 2: Font Change Karo**
```
1. Toolbar mein font dropdown click karo
2. Font select karo (Arial, Poppins, etc.)
3. Text instantly update hoga
```

### **Step 3: Size Change Karo**
```
Method 1: Toolbar se
- [-] button → Size ghatega
- [+] button → Size badhega

Method 2: Quick Actions se
- [+] button (neeche) → Size badhega
- [-] button (neeche) → Size ghatega
```

### **Step 4: Style Apply Karo**
```
1. [B] button → Bold
2. [I] button → Italic
3. [U] button → Underline
```

### **Step 5: Delete/Duplicate**
```
Delete:
- [🗑] button (neeche) click karo
- Text delete ho jayega

Duplicate:
- [⧉] button (neeche) click karo
- Copy ban jayega (offset)
```

## 🎯 Common Actions

### **Action 1: Change Font**
```
Click: Font dropdown
Select: Poppins
Result: Font changes instantly
```

### **Action 2: Make Bold**
```
Click: [B] button
Result: Text becomes bold
Click again: Text becomes normal
```

### **Action 3: Increase Size**
```
Click: [+] button (toolbar or below)
Result: Size increases by 2px
Keep clicking: Size keeps increasing
```

### **Action 4: Delete**
```
Click: [🗑] button (below element)
Result: Element deleted
Undo: Ctrl+Z to restore
```

### **Action 5: Duplicate**
```
Click: [⧉] button (below element)
Result: Copy appears (offset by 20px)
Edit: Change text in copy
```

## 🐛 Troubleshooting

### **Problem 1: Toolbar Nahi Dikh Raha**
```
Solution:
1. Text select karo (click karo)
2. Cyan border check karo
3. Toolbar upar dikhna chahiye
4. Agar nahi dikha, page refresh karo
```

### **Problem 2: Quick Actions Nahi Dikhe**
```
Solution:
1. Text select karo
2. Make sure editing mode mein nahi ho
3. Actions neeche dikhne chahiye
4. Agar nahi dikhe, deselect karke phir select karo
```

### **Problem 3: Border Blue Hai (Cyan Nahi)**
```
Solution:
1. Browser cache clear karo
2. Page refresh karo
3. Latest code pull karo
```
